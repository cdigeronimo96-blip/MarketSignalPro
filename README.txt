MarketSignalPro Coming Soon Page - Silent Signup Collection

Files included:
- index.html
- signup.php
- signups.csv

How to install on GoDaddy:
1. Open GoDaddy hosting File Manager or cPanel File Manager.
2. Go to your website root folder, usually public_html.
3. Upload index.html and signup.php.
4. Upload signups.csv too, or let signup.php create it automatically.
5. Visit https://marketsignalpro.com and test the form.

What happens when someone signs up:
- They stay on the page.
- The page shows a success message.
- A notification email is sent to support@marketsignalpro.com.
- The signup is also saved in signups.csv.

Important:
- This requires GoDaddy hosting that supports PHP.
- If your domain only has GoDaddy's "Coming Soon" builder and no hosting plan, PHP will not run.
- If email notifications do not arrive, check signups.csv first. GoDaddy may need email sending enabled or configured.
- For higher deliverability later, use Resend API instead of PHP mail().
